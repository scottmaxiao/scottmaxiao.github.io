<div ng-app="myApp" ng-controller="myCtrl">

名: <input type="text" ng-model="firstName"><br>
姓: <input type="text" ng-model="lastName"><br>
<br>
姓名: {{firstName + " " + lastName}}

</div>

<div  ng-controller="customersCtrl">
 	<ul>
  <li ng-repeat="x in names">
    {{ x }}
  </li>
</ul>
</div>


<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
    $scope.firstName= "John";
    $scope.lastName= "Doe";
});

app.controller('customersCtrl', function($scope, $http) {
    $http.get("<%=request.getContextPath()%>/template/json")
    .success(function(response) {
    	$scope.names = response;
    	});
});

</script>

AngularJS入门

介绍

    AngularJS是目前最为活跃的Javascript框架之一，AngularJS的目标之一是简化开发过程，这使得AngularJS非常善于构建小型app原型，但AngularJS对于全功能的客户端应用程序同样强大，它结合了开发简便，特性广泛和出众的性能，使其被广泛使用。

    AngularJS 诞生于2009年，由Misko Hevery 等人创建，后为Google所收购。是一款优秀的前端JS框架，已经被用于Google的多款产品当中。AngularJS有着诸多特性，最为核心的是：MVVM、模块化、自动化双向数据绑定、语义化标签、依赖注入等等。

理念

    AngularJs是一个用于设计动态web应用的结构框架。

    首先，它是一个框架，不是类库(与jQuery的区别)，是像backbone一样提供一整套方案用于设计web应用。它不仅仅是一个javascript框架，因为它的核心其实是对HTML标签的增强。

    模块化与数据的绑定，在AngularJs中，对每个功能页面都可以声明为模块，通过AngularJs的ng-model属性与模块进行数据绑定，在每个模块中定义方法或者对数据赋值，达到页面展现和数据隔离的效果。在模块中除了可以使用AngularJs提供的指令外，也可以自定义指令增强模块功能。

简单样例


<div ng-app="myApp" ng-controller="myCtrl">

名: <input type="text" ng-model="firstName"><br>
姓: <input type="text" ng-model="lastName"><br>
<br>
姓名: {{firstName + " " + lastName}}

</div>

<div  ng-controller="customersCtrl">
     <ul>
  <li ng-repeat="x in names">
    {{ x }}
  </li>
</ul>
</div>


<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
    $scope.firstName= "John";
    $scope.lastName= "Doe";
});

app.controller('customersCtrl', function($scope, $http) {
    $http.get("<%=request.getContextPath()%>/template/json")
    .success(function(response) {
        $scope.names = response;
        });
});

</script>
