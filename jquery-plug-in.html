##IOS开发笔记 ####IOS数组和字典介绍
NSArray 数组 对象有序集合
NSMutableArray 动态数组 对象有序集合,允许添加和删除数组元素
NSDictionary 字典 键值映射
NSMutableDictionary 动态字典 键值映射,允许添加和删除key和value
注：这些集合类只能容纳cocoa对象（NSOjbect对象），如果想保存一些原始的C数据（例如，int, float, double, BOOL等），则需要将这些原始的C数据封装成NSNumber类型进行存储。NSNumber对象是cocoa对象，可以被保存在集合类中。 nil不能作为空值存储到集合中，只能使用[NSNull null]存储到集合中。如 [myArray addObject:[NSNull null]]; NSArray初始化 
方式一
        NSArray *array = [[NSArray alloc] initWithObjects:@"One",@"Two",@"Three",@"Four",nil];
    方式二
        NSArray *monthNames = [NSArray arrayWithObjects:@"January",@"February",@"March",@"April",@"May",nil];
    方式三
        NSArray *array=@[@"defaultArray",@"queryArray"];
　 NSMutableArray初始化 
方式一
        NSMutableArray *array = [NSMutableArray arrayWithObjects:@"One",@"Two",@"Three",nil];
    方式二
        NSArray *array = [NSMutableArray arrayWithCapacity:20];
    [array addObject:@"One"];
    [array addObject:@"Two"];
　 NSDictionary初始化 
方式一
        NSDictionary *dictionary = [[NSDictionary alloc]initWithObjectsAndKeys:@"One",@"1",@"Two",@"2",@"Three",@"3",nil];
    方式二
        NSDictionary *parameters = @{@"search_key": self.searchBar.text,@"current_count":@"0",@"page_size":kPageSize};
　 NSMutableDictionary初始化 
方式一
        NSMutableDictionary *dictionary = [[NSMutableDictionary alloc] init];
    方式二
        NSMutableDictionary *dictionary = [NSMutableDictionary dictionary]; 
    
    方式三
        NSMutableDictionary *dictionary =[NSMutableDictionary dictionaryWithCapacity:5];
    
    //添加字典
    [dictionary setObject:@"One" forKey:@"1"];
    [dictionary setObject:@"Two" forKey:@"2"];
    [dictionary setObject:@"Three" forKey:@"3"];
    [dictionary setObject:@"Four" forKey:@"4"];
    NSLog(@"dictionary:%@",dictionary);
遍历 
数组遍历
    NSMutableArray *test = [NSMutableArray array];
    方式一
    for (int i= 0; i < test.count; i++) {
        ...
    }
    方式二
    for (id obj in test) {
        ...
    }
    
    方式三
    [test enumerateObjectsUsingBlock:^(id obj, NSUInteger idx, BOOL *stop){
        ...
    }];
    参数讲解:
    *obj:数组中的元素,该参数可以直接指定类型，如BnlogModel * model
    *idx:当前数组下标
    *stop:设为YES，就跳出循环，不继续遍历了

    ...
    }
    
    方式二
    [randomDict enumerateKeysAndObjectsUsingBlock:^(id key, id obj, BOOLBOOL *stop) {  
    ...
    }];
