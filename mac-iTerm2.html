iTerm2

iterm2是一个更好替代Mac自带终端的命令行软件。（别问我为什么是2,我也不知道。）

[下载地址：]http://iterm2.com/

快捷键介绍

Tab操作

command +t 新建一个tab

command +数字 切换tab

command + 左箭头 切换到左边的tab

command + 右箭头 切换到右边的tab

屏幕切割

command + d 屏幕垂直分割

command + shift +d 屏幕水平分割

command + w 关闭当前屏幕分割，关闭当前tab

command+] 切换到上一个屏幕分割

command+[ 切换到下一个屏幕分割

命令行

control + u 删除当前命令

command + ; 自动补全命令

command + shift +h 显示历史输入命令

其他

command + f 查找

command +q 退出iterm

ctrl +a 到行首

ctrl +e 到行尾

ctrl +h 删除光标前字符

ctrl +w 删除光标前单词
