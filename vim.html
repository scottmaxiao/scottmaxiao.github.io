#vi打开utf-8文件乱码的解决方案
set fileencodings=utf-8,gb2312,gbk,gb18030
set termencoding=utf-8
set encoding=prc
#显示行号
set nu
#取消行号
set nonu
#搜索关键字高亮

####Vim 的基本模式： 如果想正确操作上面的命令你的先理解Vim的三种模式： - 普通模式： 进入vim后最初的模式，在这种模式下面你可以输入任何不限于上面的那些编辑命令，在其他模式下只要多按几下ESC键就回到了普通模式 - 插入模式： 在普通模式下输入字符‘i, I, o, O, a, A, r, R'等插入命令，vim就进入了插入模式，在该模式下面可以插入文本字符。 - 命令行模式： 这个模式用于在编辑器底部执行命令，例如:wq等等，如果从插入模式下面切换到命令行模式或者普通模式呢，用ESC键。 ######常用快捷键
普通模式
gg文件首
G文件尾
nG到指定的行（num）。（比如 10G 就是到第 10 行）
n <enter >游標向下移動 n 行（num）。
数字0当前行首
$当前行尾
dd删除当前行
yy拷贝当前行
p粘贴

命令行模式

:q!不保存直接退出
:n到指定的行（num）。

Vim 的基本模式：

如果想正确操作上面的命令你的先理解Vim的三种模式：

  • 普通模式：

进入vim后最初的模式，在这种模式下面你可以输入任何不限于上面的那些编辑命令，在其他模式下只要多按几下ESC键就回到了普通模式

  • 插入模式：

在普通模式下输入字符‘i, I, o, O, a, A, r, R'等插入命令，vim就进入了插入模式，在该模式下面可以插入文本字符。

  • 命令行模式：

这个模式用于在编辑器底部执行命令，例如:wq等等，如果从插入模式下面切换到命令行模式或者普通模式呢，用ESC键。
